Frontend

Jest to dokumentacja struktury frontendu systemu zarządzania rezerwacjami dla osoby chcącej zaimplementować go do aplikacji desktopowej lub webowej.

Angular

Angular jest frameworkiem frontendowym wykorzystywanym w tworzeniu warstwy frontend. Działa we wsparciu z językiem TypeScript. Komunikacja z backendem odbywa się poprzez stworzone serwisy, które zawierają w sobie metody wykorzystujące wbudowane w Angulara narzędzia do komunikowania się protokołem HTTPS (metody do operacji GET, POST, PUT, PUSH, DELETE oraz interceptor do przesyłania tokenu w każdym zapytaniu).

Protokół HTTP

Wykorzystanie protokołu HTTP umożliwia wysłanie żądania HTTP i otrzymania odpowiedzi. Otrzymane później dane można przechować, a następnie wyswietlić po stronie klienta.
