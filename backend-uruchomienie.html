Procedura uruchomienia backendu




① Zainstaluj git z tego linka.

② Załóż konto na platformie Github i poproś o dostęp do repozytorium backendu programistę w zespole.

③ Otwórz systemową konsolę w folderze dla lokalizacji dla kodu i użyj poniższego polecenia 

 git clone https://github.com/Calipocalypse/web-reservation-system-backend.git .

④ Zainstaluj Visual Studio 2022. Możesz to zrobić z tego linka.

⑤ Otwórz solucję otwierając plik .sln za pomocą zainstalowanego Visual Studio.

⑥ Poczekaj, aż Visual Studio pobierze wymagane paczki (nuggety).

⑦ Otwórz konsolę "console packet manager" i użyj polecenia update-database, aby Entity Framework utworzył lokalnie bazę danych dla backendu.

⑧ Skompiluj i uruchom projekt. Twój backend w tym momencie powinien nasłuchiwać na porcie 44368.
